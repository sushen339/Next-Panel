  2. 点击打开 https://music.youtube.com/，能正常访问，代表没“送中”，反之就是“送中”了。

  4. 关闭/取消登录了谷歌账户的APP定位权限/授权；

三、如何保护自己的IP不被封锁被墙掉？

  1. 使用安全代理协议：： 加密是必备，推荐使用vless+reality+vision协议组合；
  2. 避免共享节点：： 尽量不要在不同的地区，多个省份之间不要共同连接同一个IP；
  3. 隔离IP和端口：： 避免多个用户或设备在不同地理位置漫游时使用同一个IP和端口，要分开；
  4. 控制流量：： 不要在单台VPS上长时间进行大流量下载，在一天内不要流量过高，适时切换节点；
  5. 使用高位端口：： 创建入站时，尽量使用 40000 至 65000 之间的高位端口号。
  6. 核心总结：： 不要多终端、多省份、多朋友共同使用同一个IP和端口。多创建几个入站，各用各的，避免被 GFW 识别为机场特征。 使用 Next-Panel 多创建几个【入站】， 多做几条备用，各用各的！各行其道才比较安全！ GFW的思维模式是干掉机场，机场的特征个人用户不要去沾染，自然IP就保护好了。

五、查看指定端口的网络连接数

Linux 命令：
netstat -ntu | grep :节点端口 | grep ESTABLISHED | awk '{print $5}'

  3. 可选域名（SNI）填写面板登录域名， 如果您使用其他web server（如nginx）等， 将目标网站改为对应监听端口也可；
